Bem-Vindo!


Para importar os dados de um planilha .csv para MySQL, é preciso cadastrar uma tabela no MySQL com as mesmas colunas que estão na sua planilha .csv.


Basta clicar em "Começar" e informar os dados de conexão com seu banco de dados, seja local ou não. Ao final de todo o processo será exibido uma tabela com os dados que foram importados!

Lembrando que caso a tabela tenha o campo "id", ele não será considerado (Exemplo de .sql nos arquivos).
A primeira linha do arquivo .csv também não é considerada.
